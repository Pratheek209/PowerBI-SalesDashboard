# PowerBI-SalesDashboard
Power BI dashboard project showcasing sales &amp; profit insights.
This project showcases an end-to-end Power BI dashboard analyzing sales, profit, and customer insights.

Key Features:

Sales analysis by Category, Region, and Year.

Profit trends across multiple years.

Customer segmentation by Age groups.

Quantity analysis by product category.

Interactive slicers for deep-dive filtering.

Model View + relationships created from raw dataset.

Multiple visualization types:

Bar charts.

Line charts.

Pie chart.

Small multiples.

Matrix & Cards.
